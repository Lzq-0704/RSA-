# RSA 加密算法计算器

基于 EasyX 图形库的 RSA 加密/解密桌面工具。提供图形界面进行 RSA 公密钥计算、文本加密与解密操作。

## 功能

- **计算公密钥** — 输入两个质数 p、q，自动生成 RSA 公钥 (e, n) 和私钥 (d, n)
- **加密** — 输入原文，使用公钥加密为十六进制密文，结果自动复制到剪贴板
- **解密** — 输入十六进制密文，使用私钥解密还原原文，结果自动复制到剪贴板

## 开发环境

- Visual Studio 2022 (v143)
- Windows 10 SDK
- EasyX 图形库

## 运行

打开 `RSA加密.sln`，配置 EasyX 环境后编译运行。
